Дополнительные требования к шаблонам URL (помимо стандартных требований):

  • должен начинаться строго с протоколов: http|https|*
  • содержать домен и зону без маски, либо только маску: one.com|*.one.com|*

Примеры рабочих паттернов

  • https://one.com/path*
  • *://*/*
  • https://*.one.com/path
  • https://*/path/

Работать не будут

  • https://one.*/
  • http*://one.com/
